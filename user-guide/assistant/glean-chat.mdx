## Overview

Glean Chat serves as the primary interface for interacting with Glean Assistant, offering a familiar chat-like experience similar to contemporary AI assistants. This interface enables users to engage with their company's knowledge base through natural conversations, as they would with a colleague.

<Card title="Key Features" icon="sparkles">
    • Personalized responses based on each user's permissions

    • High-confidence answers with verifiable citations

    • Natural language interaction with your organization's knowledge base
</Card>

## Accessing Glean Chat

You can access Glean Chat in two ways:

<CardGroup cols={2}>
  <Card title="Navigation Menu" icon="bars">
    Click the message bubble icon in the left-side menu of the Glean UI
  </Card>
  
  <Card title="Direct Access" icon="link" href="https://app.glean.com/chat">
    Visit the Glean Chat interface directly
  </Card>
</CardGroup>
