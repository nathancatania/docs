## Issue

- The GCP Project ID associated with the provided key isn't added to our internal manifest

## Resolution

- Please make sure that the service account key you are providing is from the same project that you would have had shared with your point of contact at Glean. If not please contact them to make sure that an entry corresponding to your GCP project has been added to our internal manifest

For additional assistance, please contact Glean Support at [https://support.glean.com](https://support.glean.com)
