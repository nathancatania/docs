## Issue:

The associated Google Drive datasource has a configuration conflict, which will prevent crawls from starting.

## Resolution:

- Make sure that the selected associated Google Drive connected instance is selected for atmost one Google Chat instance.
- If the problem persists, please ask a glean representative to fix it.

If you are still having difficulty please reach out to us: [https://support.glean.com](https://support.glean.com)
