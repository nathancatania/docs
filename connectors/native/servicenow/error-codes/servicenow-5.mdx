## Issue

- Unable to access the advanced user criteria REST endpoint /user_criteria

## Resolution

- Validate that the create GleanSearch scripted API has been created and configured as per the instructions for Advanced setup
- Ensure the ACL to use that scripted API is granted to the service account user
- Ensure the provided API Namespace configuration matches with the created GleanSearch scripted API
