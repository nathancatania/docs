title: SALESFORCE_1
description: SALESFORCE_1 Error Code
---

## Issue

- This error shows up when we are not able to retrieve the access permissions available to the authenticated user.

## Resolution:

- This error only shows up when the authenticated user is not using a System Administrator profile.
- Ensure that the permissions mentioned in this article here are set correctly. Specifically,
    - Ensure the `View Roles and Role Hierarchy` permission is set.
    - Ensure the `View Setup and Configuration` permission is set.
