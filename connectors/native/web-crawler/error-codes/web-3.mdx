---
title: WEB_3
description: WEB_3 Error Code
---

Both sitemap and seed URLs are provided

Please ensure that either a sitemap URL is provided or a list of seed URLs, but not both. Multiple seed URLs should be separated by commas for example seeds = `url1,url2,url3`

For additional assistance, please contact Glean Support at [https://support.glean.com](https://support.glean.com)
