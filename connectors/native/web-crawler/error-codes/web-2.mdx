---
title: WEB_2
description: WEB_2 Error Code
---

Custom datasource name config is not lowercase and alphanumeric

Please ensure that the custom datasource name config consists of lowercase alphabets and decimal digits only. For example web2qs9yszgleanhelp is a valid custom datasource name

For additional assistance, please contact Glean Support at [https://support.glean.com](https://support.glean.com)
