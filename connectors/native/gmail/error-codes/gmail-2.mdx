## Issue

Could not connect to Gmail service.

## Resolution

- GMAIL uses the Directory Admin configured for the GDRIVE App. Ensure a Directory Admin e-mail is set in the GDRIVE App and that this is a valid email account on the Gmail service.
- Ensure the Gmail Service is active.
- Ensure that correct scopes have been granted.

For additional assistance, please contact Glean Support at [https://support.glean.com](https://support.glean.com)
