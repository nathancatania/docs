title: O365-13
description: O365-13 Error Code
---

## Issue:

- A url you added to the Sharepoint Site Url list is incorrect. This list should contain site and subsite urls.

## Resolution:

Please update the invalid url.

Valid examples:

- `https://foo.sharepoint.com/sites/site_name`
- `https://foo.sharepoint.com/sites/site_name/subsite_name`


Invalid examples:

- `https://foo-my.sharepoint.com/personal/username_foo_com`
- `https://foo.sharepoint.com/sites/site_name/SitePages`

For additional assistance, please contact Glean Support at [https://support.glean.com](https://support.glean.com)
