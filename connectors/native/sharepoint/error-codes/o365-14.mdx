title: O365-14
description: O365-14 Error Code
---

## Issue:

- The client secret used for the Onedrive / Sharepoint integration is about to expire.

## Resolution:

Navigate to the Onedrive/Sharepoint application setup page in the workspace settings page.

- Have the tenant administrator navigate to the App Registrations page.
- As per the application setup instructions, go through the steps to Generate Secret.
- Input and save the new client secret created in the "Client Secret" box. Ensure this is the client secret, not the client value.

For additional assistance, please contact Glean Support at [https://support.glean.com](https://support.glean.com)
