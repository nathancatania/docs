## Issue

The directory admin account you entered is missing privileges to list shared drives.

## Resolution

- Verify that the directory admin email you entered is valid and is an admin who has access to shared drives.
- If you have created a custom admin role, verify that the role you created has the following "Admin Console Privilege": **Services > Drive and Docs > Settings**

If you are still having difficulty please reach out to us: [https://support.glean.com](https://support.glean.com)
