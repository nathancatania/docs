## Issue

- Page content redlist regex entered is invalid.

## Resolution:

- Please ensure that the regex is valid. Any valid regex is accepted. For example, if you require to redlist all pages with the word "sensitive" in it, then the regex could be:
`.*\b(sensitive)\b.*`

If you are still having difficulty, please reach out to us: [https://support.glean.com](https://support.glean.com)
