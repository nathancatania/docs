## Issue

- The list provided in Default access group field contains invalid entries or is empty.

## Resolution:

- Please check whether each of items of the list:
  - Is a valid product access group
  - Contains at-least one user.
- The membership of a product access group can be checked via admin.atlassian.com ([reference](https://support.atlassian.com/user-management/docs/how-does-product-access-work/))

If you are still having difficulty, please reach out to us: [https://support.glean.com](https://support.glean.com)
