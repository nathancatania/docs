## Issue

We were unable to create a vault export via the Google Groups API.

## Resolution

Please ensure that the configured directory domain user has access to the provided matter ID. Additionally ensure the Vault API is enabled.

For additional assistance, please contact Glean Support at [https://support.glean.com](https://support.glean.com)
