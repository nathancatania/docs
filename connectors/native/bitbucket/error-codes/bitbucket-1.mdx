# Issue

We are unable to fetch Bitbucket workspaces using the OAuth consumer credentials that were provided.

# Resolution

### For Bitbucket Cloud:

Please ensure that the Bitbucket OAuth consumer has access to the workspaces that should be crawled (there must be at least one) and was given the permissions "Account:Read" and "Account:Email".

### For Bitbucket Server:

This error is not applicable.

If you are still having difficulty please reach out to us: [https://support.glean.com](https://support.glean.com).
