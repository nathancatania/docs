---
title: ATLASSIAN_1
description: ATLASSIAN_1 Error Code
---

## Issue

- For Datacenter apps, the base URL entered is in an incorrect format. The Base URL should not be empty and should not end with a "/" (slash)
- For Cloud apps, the Atlassian domain entered is in an incorrect format.

## Resolution

- For Datacenter apps, please enter the base URL in the correct format.
- For Cloud apps, please enter the Atlassian domain in the correct format. If your Atlassian instance url looks like `https://company.atlassian.net`, then your Atlassian domain is `company`.

If you are still having difficulty please reach out to us: [https://support.glean.com](https://support.glean.com)
