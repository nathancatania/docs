## Issue:

- The Forge app creator credentials are not valid. It is possible that the credentials are empty, invalid or expired.

## Resolution:

- Please ensure you have entered the Forge app creator credentials correctly. You can create a API token for the Forge app creator account from [https://id.atlassian.com/manage-profile/security/api-tokens](https://id.atlassian.com/manage-profile/security/api-tokens).

If you are still having difficulty please reach out to us: [https://support.glean.com](https://support.glean.com)
