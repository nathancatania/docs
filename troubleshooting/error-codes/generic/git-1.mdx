## Issue

The VM used to process git repository content (files, directories, readmes, pages) is not reachable by the crawler.

## Resolution

Please ensure that the Proxy IP address in the Glean Transit VPC has appropriate routes and security policy in place to reach the VM.

For additional assistance, please contact Glean Support at [https://support.glean.com](https://support.glean.com)
